# EchoNote 🎙️✨

EchoNote is an offline-first, accessibility-first iOS mobile application engineered to assist neurodivergent and hard-of-hearing individuals with live auditory intake retention. The system captures environmental sound fields, structures them locally using on-device Natural Language Parsing, and maps them onto a customizable, highly scannable real-time typographic interface.

Built explicitly for the **Apple Developer Academy** application portfolio, this utility prioritizes platform integrity, human-centered accessibility design metrics, and optimization for Apple Silicon.

## 🛠️ Deep Native Tech Stack
* **UI Interface Engine:** Declarative SwiftUI layout frameworks.
* **Local Identity Storage:** SwiftData persistent architecture mappings.
* **Audio Synthesis Interface:** Native `AVFAudio` processing layers (`AVAudioEngine`).
* **Linguistic Translation Layer:** On-Device Speech Framework models (`SFSpeechRecognizer`).
* **Semantic Analysis Pass:** Apple Natural Language Framework (`NLTokenizer`).
* **Core Concurrency Foundation:** Safe Swift 5.10+ async streams and Actor-isolated context routines.
